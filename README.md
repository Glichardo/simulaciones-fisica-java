# Simulaciones físicas

> **Códigos sencillos en Java para simular caídas, péndulos y más.**   

---

## Funcionamiento

Usa fórmulas físicas para simular distintos fenómenos. De momento solo se puede ejecutar en la consola del IDE

---

## Problemas conocidos

- Se producen distintas excepciones cuando los datos introducidos no tienen sentido. Será arreglado en el futuro cuando limpie un poco más el código, de momento solo quiero que funcione.
- Los bugs y fallos estarán subidos en la pastaña de "Issues" de GitHub
